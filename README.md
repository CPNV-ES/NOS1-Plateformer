# Projet Scolaire de Jeu de Plateforme

## Statut du projet : Terminé

Ce projet a été réalisé dans le cadre d'un travail scolaire et est maintenant considéré comme terminé. L'objectif était de prendre un jeu de plateforme existant et d'y intégrer plusieurs fonctionnalités, notamment une base de données pour la gestion des scores.

Toutes les fonctionnalités prévues ont été implémentées et le projet a été soumis pour évaluation.

### Contributions

**Nous n'acceptons pas de contributions ou de "pull requests" sur ce projet.** Étant donné qu'il s'agit d'un projet scolaire finalisé, nous ne prévoyons pas d'y apporter de modifications ou d'ajouts.

### Jouer au jeu

Si vous souhaitez jouer à la version fonctionnelle du jeu et enregistrer votre score, veuillez télécharger la version `v1` disponible dans les "[releases](https://github.com/CPNV-ES/NOS1-Plateformer/releases/tag/v1)". Cette version inclut la configuration nécessaire pour se connecter à la base de données, qui n'est pas présente dans le code source de cette branche pour des raisons de sécurité.
